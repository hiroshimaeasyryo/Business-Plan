# 2027年9月期 事業計画に向けた損益分岐点分析の進め方

## 1. 目的

2027年9月期の事業計画策定にあたり、目標利益から逆算して必要売上高を定める。
そのために、今期実績をベースに以下を事業部別に設計する。

- 必要売上高
- 売上原価
- 売上総利益
- 販管費
- 営業利益
- 営業外利益
- 営業外費用
- 経常利益

あわせて、以下の論点を整理する。

- 固変分解をどの定義で行うか
- どの費目までを分解対象とするか
- 事業部ごとの利益体質の違いをどう比較するか
- 本部費用をどのように配賦、または別管理するか

## 2. 先に結論: 実務で使いやすい基本方針

事業計画で使う損益分岐点分析は、まずは **営業利益ベース** を中心に置くのが実務的。

理由:

- 損益分岐点分析は本来、売上と営業活動コストの関係を見る道具である
- 営業外損益は売上増減と連動しないことが多く、固変分解との相性が弱い
- 事業部への予算割当は、まず事業活動そのものの収益性で判断した方がブレが少ない

したがって、基本モデルは次の2段階に分ける。

### 第1段階: 営業利益までを固変分解の主対象にする

- 売上高
- 変動売上原価
- 固定売上原価
- 変動販管費
- 固定販管費
- 営業利益

### 第2段階: 営業外損益を加えて経常利益計画へ接続する

- 営業外利益
- 営業外費用
- 経常利益

営業外損益は原則として、固変分解ではなく **別建て計画** にする。

## 3. 固変分解の対象範囲の考え方

### 推奨範囲

固変分解の対象は、原則として **営業利益まで**。

対象に含めるもの:

- 売上高
- 売上原価
- 販管費

対象に含めないもの:

- 営業外利益
- 営業外費用
- 特別利益
- 特別損失
- 法人税等

### 含めない理由

- 営業外損益は金融収支、補助金、雑収入など売上との因果が弱い
- 特別損益は一過性であり、来期の経営意思決定指標にしにくい
- 税金は結果として決まるため、事業部評価のKPIには不向き

### 例外

もし特定事業部において営業外費用が事業構造と強く結びつく場合は、管理用の補助指標として見る。
ただし損益分岐点の基本モデルには混ぜない方がよい。

## 4. どのような固変分解モデルを採用するか

実務上は、まず **準変動費を含む3区分の考え方** を持ちながら、計画モデル上は2区分に落とすのが使いやすい。

### モデルA: シンプルな2区分モデル

- 変動費
- 固定費

最終的な損益分岐点分析の計画式:

`営業利益 = 売上高 - 変動費 - 固定費`

または

`営業利益 = 限界利益 - 固定費`

このモデルの利点:

- 説明しやすい
- 事業部別比較がしやすい
- 予算編成にそのまま使いやすい

### モデルB: 実務判定用の3区分モデル

- 変動費
- 準変動費
- 固定費

準変動費の例:

- 稼働量に応じて増減する外注費
- 売上に完全比例しない人件費
- ある閾値を超えると増える物流費

使い方:

- 実績分析時には3区分で見る
- 予算モデルへ落とすときは、準変動費を変動部分と固定部分に再配分して2区分化する

## 5. 費目ごとの基本判定ルール

事業部別に費目を以下の観点で判定する。

### 変動費

売上高、取扱高、案件数、稼働時間、出荷件数などに応じて増減する費用。

例:

- 仕入原価
- 販売手数料
- 支払ロイヤリティ
- 配送費の一部
- 外注費の一部

### 固定費

短期的には売上の増減にかかわらず発生する費用。

例:

- 固定給人件費
- 地代家賃
- システム利用料
- 減価償却費
- 本部管理人件費

### 要検討費目

一律に決めず、事業部ごとに判定すべき。

- 人件費
- 外注費
- 広告宣伝費
- 旅費交通費
- 通信費
- 業務委託費

## 6. 事業部別に利益体質を比べるための見方

事業部ごとに見るべきKPIは次の通り。

### 1. 限界利益率

`限界利益率 = (売上高 - 変動費) / 売上高`

高いほど、売上増が利益につながりやすい。

### 2. 固定費回収力

`固定費回収力 = 限界利益 - 固定費`

営業利益そのものでもよいが、まずは限界利益で固定費をどれだけ吸収できるかを見る。

### 3. 損益分岐点売上高

`損益分岐点売上高 = 固定費 / 限界利益率`

低いほど、利益が出やすい体質。

### 4. 安全余裕率

`安全余裕率 = (実績売上高 - 損益分岐点売上高) / 実績売上高`

高いほど、売上変動に耐えやすい。

### 5. 営業利益率

通常のP/L管理との接続用。

## 7. 事業部ごとに予算配分へつなぐ考え方

予算配分は、単に前年踏襲ではなく、少なくとも次の3軸で考える。

- 市場成長性
- 限界利益率の高さ
- 固定費増分に対する利益回収の速さ

考え方の例:

- 限界利益率が高く、固定費追加で売上を伸ばせる事業部には積極配分
- 限界利益率が低く、固定費先行型の事業部には慎重配分
- 戦略投資が必要な事業部は、短期採算とは分けて投資枠で管理

ここで重要なのは、**収益事業の予算** と **戦略投資の予算** を混ぜないこと。

## 8. 本部費用の扱い

本部費用は、意思決定目的に応じて **2本立て** で管理するのがよい。

### 見方A: 事業部の実力を見るための管理損益

本部費配賦前の事業部利益を見る。

目的:

- 事業そのものの稼ぐ力を測る
- 事業部長の責任範囲を明確にする

### 見方B: 全社損益につなぐための配賦後損益

本部費を一定ルールで配賦した後の利益を見る。

目的:

- 全社計画に接続する
- 経常利益計画に整合させる

### 配賦ルールの候補

- 売上高基準
- 粗利基準
- 人員数基準
- 工数利用基準
- システム利用量基準

### 推奨

まずは以下の2層に分ける。

- 直接帰属できる本部費: 各事業部に直課
- 共通本部費: 配賦ルールで按分

そして、配賦は一種類に固定せず、費目ごとに合理的な基準を使う。

例:

- 情報システム費: 利用ID数
- 採用人事費: 人員数
- 経営管理費: 売上または粗利

## 9. 実務での進め方

### Step 1. 事業部別の実績P/Lを並べる

最低限必要な列:

- 事業部
- 売上高
- 売上原価の内訳
- 販管費の内訳
- 営業外利益
- 営業外費用

### Step 2. 各費目を固変判定する

各費目に以下の属性を付ける。

- 勘定科目
- 事業部
- 発生部門
- 変動 / 固定 / 準変動
- 判定根拠
- 連動ドライバー

### Step 3. 事業部別に限界利益率を算出する

ここで各事業部の儲けやすさの一次評価を行う。

### Step 4. 本部費の直課分と共通分を分ける

そのうえで、共通分の配賦ルールを設計する。

### Step 5. 来期前提を置く

事業部ごとに以下を設定する。

- 売上成長率
- 変動費率
- 固定費増減
- 追加投資
- 営業外損益見込み

### Step 6. 目標利益から逆算する

たとえば経常利益目標を置く場合:

1. 経常利益目標
2. 営業外損益計画を加味して必要営業利益を逆算
3. 固定費計画から必要限界利益を逆算
4. 限界利益率から必要売上高を逆算

最低でも次の3表を作ると進めやすい。

### 1. 事業部別実績P/L一覧

### 2. 費目別固変判定表

列例:

- 科目
- 金額
- 区分
- 配賦有無
- 直課/共通
- ドライバー

### 3. 事業部別採算モデル

列例:

- 売上高
- 変動費
- 限界利益

- 固定費
- 営業利益
- 損益分岐点売上高
- 安全余裕率
- 配賦前利益
- 配賦後利益

## 11. 壁打ちの進め方

最初の壁打ちは、次の順がよい。

1. 事業部の切り方を確定する
2. 実績P/Lの費目粒度を揃える
3. 固変分解の基準を仮置きする
4. 本部費の直課/共通を分ける
5. 事業部別の限界利益率と固定費構造を比較する
6. 予算配分の考え方を決める

## 12. 現時点の推奨方針

現時点では、以下の方針で始めるのが最も実務的。

- 固変分解の主対象は営業利益まで
- 営業外損益は別建てで計画
- 実績分析では準変動費を認識しつつ、予算モデルは2区分化
- 事業部比較は限界利益率と固定費回収力を軸に行う
- 本部費は配賦前と配賦後の両方を見る

## 13. 次に確認したいこと

次の情報が揃うと、かなり具体化できる。

- 事業部の数と名称
- 各事業部の売上構造の違い
- 今期実績P/Lの費目粒度
- 人件費、外注費、広告費の扱い
- 本部費の主な内訳
